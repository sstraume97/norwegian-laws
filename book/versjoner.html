Stabile versjoner

Hver årsversjon (v{årstall}) er et øyeblikksbilde av alle norske lover slik de var ved utgangen av det året, basert på law-history-grenen.

Verktøy

  • Søk i lover — finn lover etter tittel eller korttittel
  • Sammenlign lovversjon — velg lov og to årstall for å se endringer

Bruk med git

# Klon historikk-grenen
git clone -b law-history https://github.com/sondreskarsten/norwegian-laws.git
cd norwegian-laws

# Se en lov slik den var i 2020
git show v2020:lover/lov-1998-07-17-56.md

# Sammenlign to versjoner av en lov
git diff v2020 v2024 -- lover/lov-1998-07-17-56.md

# Se alle endringer mellom to år
git diff --stat v2023 v2024
