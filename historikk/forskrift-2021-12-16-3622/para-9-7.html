Gjeldende tekst — § 9-7. Fartsgrenser på sjøen i Tvedestrand kommune

I perioden 1. mai–31. august er 5 knop høyeste tillatte fart for næringsfartøy innenfor 50 meter fra land, øyer, holmer og skjær som er synlige over vannflaten dersom det ikke er bestemt noe annet.

5 knop er høyeste tillatte fart hele året for næringsfartøy i følgende områder:

  • Indre havn innenfor linje trukket mellom posisjonene N 58° 37.116′, Ø 8° 56.143′ (Bakkeskåt) N 58° 37.231′, Ø 8° 56.367′ (ved Tangen)
  • Kvastadkilen innenfor linje trukket mellom posisjonene N 58° 35.546′ N, Ø 8° 56.667′ (ved Høyvika) N 58° 35.552′ N, Ø 8° 56.825′ (Holmen)
  • Røskilen innenfor linje trukket mellom posisjonene N 58° 35.581′, Ø 8° 57.132′ (Holmen) og N 58° 35.580′, Ø 8° 57.310′ (ved Ramsdalen)
  • Sandøykilen innenfor linje trukket mellom posisjonene N 58° 35.928′, Ø 9° 03.328′ (vestsiden av innløpet til Sandøykilen) N 58° 35.941′, Ø 9° 03.365′ (østsiden av innløpet til Sandøykilen)
  • Snaresund innenfor en linje trukket mellom posisjonene N 58° 35.063′, Ø 8° 59.431′ (Groviktangen) N 58° 35.037′, Ø 8° 59.545′ (Bukkholmen vest) og N 58° 35.037′, Ø 8° 59.694′ (Bukkholmen øst) N 58° 34.923′, Ø 8° 59.926′ (Kalvenlandet) og N 58° 35.719′, Ø 9° 00.594′ (Tåknes) N 58° 35.577′, Ø 9° 00.632′ (Valholmen nordøst) og N 58° 35.505′, Ø 9° 00.565′ (Borøy)
  • Gjeving Nybrygga innenfor linje trukket mellom posisjonene N 58° 38.274′, Ø 9° 06.587′ (fastlandet innenfor Tørkersholmene), N 58° 38.329′, Ø 9° 06.731′ (Hellersøya sørvest) og N 58° 38.479′, Ø 9° 07.087′ (Hellersøya øst) N 58° 38.522′, Ø 9° 07.324′ (Mannholmen sør) N 58° 38.614′, Ø 9° 07.770′ (fastlandet sørøst for Gjeving)
  • Området Store og lille Svalsund, Lyngørsundet og Håkonsund innenfor linje trukket mellom posisjonene N 58° 37.671′, Ø 9° 06.515′ (Håkonsund vest) N 58° 37.698′, Ø 9° 06.598′ (Håkonsund øst) og N 58° 38.130′, Ø 9° 07.267′ (Steinsøya) N 58° 38.187′, Ø 9° 07.424′ (Holmen) og N 58° 38.215′ N, Ø 9° 07.572′ (Holmen) N 58° 38.251′ N, Ø 9° 07.691′ (Odden) og N 58° 38.337′, Ø 9° 08.345′ (Odden) N 58° 38.240′, Ø 9° 08.564′ (Lyngørsiden) og N 58° 37.651′, Ø 9° 07.540′ (Østpynten) N 58° 37.706′, Ø 9° 07.651′ (Ringpynten)
  • Risøya og området innenfor, innenfor linjer trukket mellom posisjonene N 58° 39.024′, Ø 9° 08.482′ (Rota) N 58° 39.047′, Ø 9° 08.629′ (Risøya) til kommunegrensen mot Risør
  • Lagelva, ved innløpet innenfor linje trukket mellom posisjonene N 58° 40.693′, Ø 9° 02.832′ N 58° 40.716′, Ø 9° 02.954′ og N 58° 41.072′, Ø 9° 04.328′ N 58° 41.043′, Ø 9° 04.509′.

I perioden 1. mai–31. august er 5 knop høyeste tillatte fart for fritidsfartøy i den delen av hoved- og biled som ligger innenfor 50 meter fra land, øyer, holmer og skjær som er synlige over vannflaten dersom det ikke er bestemt noe annet.

5 knop er høyeste tillatte fart hele året for fritidsfartøy i den delen av områdene Store og Lille Svalsund, Lyngørsundet og Håkonsund som er hoved- og biled innenfor en linje trukket mellom posisjonene og og og og

  • N 58° 37.671′, Ø 9° 06.515′ (Håkonsund vest)
  • N 58° 37.698′, Ø 9° 06.598′ (Håkonsund øst)
  • N 58° 38.130′, Ø 9° 07.267′ (Steinsøya)
  • N 58° 38.187′, Ø 9° 07.424′ (Holmen)
  • N 58° 38.215′ N, Ø 9° 07.572′ (Holmen)
  • N 58° 38.251′ N, Ø 9° 07.691′ (Odden)
  • N 58° 38.337′, Ø 9° 08.345′ (Odden)
  • N 58° 38.240′, Ø 9° 08.564′ (Lyngørsiden)
  • N 58° 37.651′, Ø 9° 07.540′ (Østpynten)
  • N 58° 37.706′, Ø 9° 07.651′ (Ringpynten).
