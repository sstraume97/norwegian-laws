Forskrift om regulering av fisket etter dyphavsarter i NEAFC' reguleringsområde i 2026

§ 1. Generelt forbud

Det er forbudt for norske fartøy å fiske og lande følgende arter i NEAFC' reguleringsområde i 2026:

  • Argentines ( Argentina spp.)
  • Ålekvabbe ( Zoarces viviparus )
  • Skjellbrosme ( Phycis blennoides )
  • Bairds's smoothhead ( Alepocephalus bairdii )
  • Risso's smoothhead ( Alepocephalus rostratus )
  • Blå antimora ( Antimora rostrata )
  • Dolkfisk ( Aphanopus carbo )
  • Vassild ( Argentina silus )
  • Beryx ( Beryx spp.)
  • Brosme ( Brosme brosme )
  • Dypvannsrødkrabbe ( Chacon (Geyron) affinis )
  • Havål ( Conger conger )
  • Svart (dypvanns) kardinalfisk ( Epigonus telescopus )
  • Blåkjeft ( Helicolenus dactylopterus )
  • Silver roughy (pink) ( Hoplostethus mediterraneus )
  • Reimfisk (slirefisk) ( Lepidopus caudatus )
  • Ulvefisk ( Lycodes esmarkii )
  • Blålange ( Molva dypterigia )
  • Lange ( Molva molva )
  • Mora ( Mora moro )
  • Flekkpagell ( Pagellus bogaraveo )
  • Forkbeards/Skjellbrosme ( Phycis spp.)
  • Vrakfisk ( Polyprion americanus )
  • Blåkveite ( Reinhardtius hippoglossoides )
  • Lusuer ( Sebastes viviparus )
  • Skolestfamilien ( Macrouridae )
  • Orange roughy ( Hoplostethus atlanticus )
  • Spiny scorpionfish ( Trachyscorpia cristulata )
  • Straightnose rabbitfish ( Rhinochimaera atlantica )
  • Blåvinget havmus ( Hydrolagus mirabilis )
  • Havmus ( Chimaera monstrosa )
  • Rundskate ( Raja fyllae )
  • Isskate ( Raja hyperborea )
  • Svartskate ( Raja nidarosienus )
  • Iceland catshark ( Apristuris spp.)
  • Gulper shark ( Centrophorus granulosus )
  • Brunhå ( Centrophorus squamosus )
  • Islandshå ( Centroscyllium fabricii )
  • Dypvannshå ( Centroscymnus coelolepis )
  • Bunnhå ( Centroscymnus crepidater )
  • Frilled shark ( Chlamydoselachus anguineus )
  • Spansk håkjerring ( Dalatias licha )
  • Gråhå ( Deania calceus )
  • Stor svarthå ( Etmopterus princeps )

  • Hågjel ( Galeus melastomus )
  • Mouse catshark ( Galeus murinus )
  • Lantern sharks ( Etmopterus spp.)
  • Kamtannhai ( Hexanchus griseus )
  • Tornhå ( Oxynotus paradoxus )
  • Knifetooth dogfish ( Scymnodon ringens )
  • Håkjerring ( Somniosus microcephalus )
  • Opal chimaera ( Chimaera opalescens )
  • Smallspine spookfish ( Harriotta haeckeli )
  • Narrownose chimaera ( Harriotta raleighana )
  • Small-eyed rabbitfish ( Hydrolagus affinis )
  • Portuguese rabbitfish ( Hydrolagus lustanicus )
  • Pale chimaera ( Hydrolagus pallidus )
  • Brugde ( Cetorhinus maximus )
  • Håbrann ( Lamna nasus ).

§ 2. Adgang til å fiske visse arter

Uten hinder av forbudet i § 1 kan norske fartøy fiske i NEAFC' reguleringsområde etter artene i § 1 nr. 1–26 og 28.

§ 3. Forbud mot å fiske med bunnredskap

Det er ikke tillatt å fiske med bunnredskap (bunntrål, bunnline og bunngarn) i perioden 15. februar til 15. april innenfor et område avgrenset av rette linjer mellom følgende posisjoner:

  • N 60° 58′ 76 V 27° 27′ 32
  • N 60° 56′ 02 V 27° 31′ 16
  • N 60° 59′ 76 V 27° 43′ 48
  • N 61° 03′ 00 V 27° 39′ 41.

§ 4. Forbud mot fiske i et område ved Rockall

Det er ikke tillatt å fiske med andre redskap enn line innenfor et område avgrenset av rette linjer mellom følgende posisjoner:

  • N 57° 00′ V 15° 00′
  • N 57° 00′ V 14° 00′
  • N 56° 30′ V 14° 00′
  • N 56° 30′ V 15° 00′.

§ 5. Garnforbud

Det er forbudt å sette garn dypere enn 200 meter.

§ 6. Utkastforbud

Det er ikke tillatt å kaste ut fisk nevnt i § 1 som er død eller døende.

§ 7. Bifangst av hai

Dersom det tas hai som bifangst i andre fiskerier, skal disse søkes levende gjenutsatt.

§ 8. NEAFC-reguleringer

Fartøy skal følge regler fastsatt i eller i medhold av Konvensjon om det fremtidige multilaterale samarbeid om fisket i det nordøstlige Atlanterhav (NEAFC-konvensjonen).

§ 9. Straff

Den som forsettlig eller uaktsomt overtrer bestemmelser gitt i eller i medhold av forskriften, straffes i henhold til havressurslova §§ 60 , 61 , 64 og 65 . På samme måte straffes forsøk og medvirkning.

§ 10. Ikrafttredelse

Forskriften trer i kraft 1. januar 2026 og gjelder til og med 31. desember 2026.
